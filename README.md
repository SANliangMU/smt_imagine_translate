# SMT_IMAGINE_TRANSLATE

#### 介绍

SMT: IMAGINE 的汉化翻译文本仓库。

#### 文件格式

| source | translate | information |
| --- | --- | --- |
| 日文原文 | 翻译文本 | 翻译相关信息 |

由于文件格式为 csv 格式，采用英文逗号进行列数据分割，所以原文中的英文逗号采用 %2C 进行替换。

日文原文中存在多行数据，采用 \n 文本标记换行。

由于技术不足，尽量采用日文汉字进行文本翻译。

无需翻译的原文可以整行删除。
